## 🇬🇧 English

Hi! This is a small full-stack project I built after finishing my 1st year at UNITBV (AIA). I wanted to practice Object-Oriented Programming in Java and see how it can communicate with a web interface.

The app lets you track expenses and incomes. Instead of using a database, it saves everything locally on your PC in a simple text file, processing the data through Java.

### 🛠️ How it's built
* **Backend:** Written in Java. It uses the native `HttpServer` (no heavy external libraries needed) and handles data saving using File I/O.
* **Frontend:** A clean, dark-mode webpage made with HTML, CSS, and vanilla JavaScript (`Fetch API` to talk to the Java server).

### 🚀 How to run it
1. Run `Main.java` in your IDE. This will start the local server on port `7075`.
2. Open the `web/index.html` file in your browser. 
3. Click "Actualizează" to see the balance update in real-time straight from the Java backend.
